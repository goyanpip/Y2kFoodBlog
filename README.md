🌈 個人佳牌冰箱布丁部落格 (v1.0.2-BETA) 🍮
歡迎來到個人佳牌冰箱布丁的官方原始碼倉庫！這是一個致敬 2000 年代網頁設計巔峰（或者是混亂巔峰）的實驗性專案。這裡不歡迎米其林大廚，我們只歡迎半夜肚子餓且冰箱剩菜一堆的勇者。

✨ 視覺特效特點 (Web Features)
新細明體 (PMingLiU)：純正的 Windows 98/XP 視覺魂。

跑馬燈 (Marquee)：標題會飛過來又飛過去，讓你目不暇給。

極度密集排版：拒絕留白！資訊量必須大到讓你視覺疲勞。

GIF 大量發生：收集自 GifCities 的經典動圖，閃爍、旋轉、跳躍。

彩虹滑鼠軌跡：引入 cursor-effects 庫，滑鼠動到哪，彩虹就跟到哪。

芒果與水花飄飄：自帶隨機落下的 🥭 與 💦 粒子特效，打造夢幻又莫名其妙的網頁動效。

🛠️ 食譜內容 (Included Recipes)
OMG Soup：心臟不好的人請勿隨便打開鍋蓋。

小潘偉蕙跳水蛋糕：一場重力與乳製品的悲劇。

冰箱雞蛋毛炒餅：利用肉鬆與硬掉餅皮進行的植毛工程。

果仁冰箱布丁：部落格的看板娘料理，吸取了冰箱靈魂的預言甜點。

🚀 如何架設 (Deployment)
下載 index.html。

不要用任何現代的排版引擎去修改它。

直接用瀏覽器開啟（推薦使用 Internet Explorer... 喔不，現在用 Chrome/Edge 也可以，但請想像自己在用 IE 6.0）。

享受那種螢光綠文字刺痛雙眼的快感。
